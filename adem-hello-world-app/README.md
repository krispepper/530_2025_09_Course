# ADEM Hello World Web App (Node.js + MongoDB)

## Description
This is a simple Hello World web app for the M3 Hello World Coding assignment.  
It shows use of a Node.js + Express backend with a MongoDB database.  
---

## Setup Instructions (for compsci server)

### 1. SSH into the compsci server
ssh compsci.adelphi.edu

### 2. After login, Move into the shared group project folder
cd /home/groupprojects/fall2025_530_course/adem-hello-world-app

### 3. Install dependencies and start the app
npm install
npm start
